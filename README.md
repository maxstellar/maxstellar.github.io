# maxstellar.github.io
my personal site! once private, now made public so i can ship this to HCTG
i was told to make this readme detailed so here we go!
built entirely in HTML, CSS, and JS, relying on zero libraries or frameworks, this personal website is a simple one page design that has multiple 100vh sections, so each section feels like its own page. i chose an accent color of pale yellow because it is my favorite. i wrote about my own life, my experiences, and my projects. i tried many different things to make this website come to life, like adding images, videos, interactive elements, but at the end of the day, i just had to be true to myself and write about who i was. the moral of the story is, never change who you are for 6 (or less) HCTG tickets. the end

to whoever is reviewing this... it's not organized. my personal has never been organized and it's tradition to keep it that way. i'm very sorry

## ai disclosure
ai was used for the background (twinkling stars effect) + the js listener that triggers buttons to glow when their section is scrolled to, and that's it!
